## ミッション

**エビデンスに基づく · ハームリダクション · 医療へのガイド**

HRT薬典は、中国語圏のトランスジェンダー女性を対象としたHRT安全基準情報サイトです。医師の代わりになることが目的ではなく、すでにホルモン療法を行っている、またはこれから始める方々を支援します：

- **血液検査の結果を正しく理解**し、どの異常値で受診すべきかを把握する
- **薬のリアルなリスクを知る** — コミュニティの噂や未検証の体験談ではなく
- **緊急の危険信号を認識する**（DVT、肝障害、髄膜腫）— 重要な場面で正しい判断を下すために
- **トランスフレンドリーな医療リソースを見つける** — DIYから正規の医療へ

### このサイトが提供しないもの

- ❌ 百科事典や医学教科書ではありません
- ❌ フォーラム、コミュニティ、交流プラットフォームではありません
- ❌ 薬の購入ルートや推奨サイトではありません
- ❌ 個別の処方アドバイスは提供しません（「あなたはXmg服用すべき」など）

## コンテンツの出典

本サイトのすべての医療コンテンツは、以下の国際臨床ガイドラインおよび査読済み文献に基づいています：

| 出典 | 説明 |
|------|------|
| **Endocrine Society 2017** | 《Endocrine Treatment of Gender-Dysphoric/Gender-Incongruent Persons》臨床実践ガイドライン |
| **WPATH SOC 8 (2022)** | 《Standards of Care for the Health of Transgender and Gender Diverse People》第8版 |
| **UCSF Transgender Care** | カリフォルニア大学サンフランシスコ校トランスジェンダーケアガイドライン |
| **EMA (2020)** | 欧州医薬品庁によるCPAの使用制限および安全性評価 |
| **PubMed / PMC** | 査読済みのオリジナル研究論文（各引用にDOI付き） |

各医療推奨の横にある上付き数字（例：<sup>[1]</sup>）は、具体的な文献にリンクしています。**引用なし＝コンテンツなし** — これは本サイトの鉄則です。

## コンテンツレビュープロセス

| 項目 | 基準 |
|------|------|
| **引用要件** | すべての臨床推奨は、少なくとも1つの査読済み論文または国際ガイドラインに遡及可能であること |
| **クロスバリデーション** | 薬剤の投与量範囲は2つ以上の独立した情報源と照合（例：WPATH＋Endocrine Society＋UCSF） |

## プライバシー

**本サイトは個人データを一切収集・保存しません。**

| カテゴリ | 対策 |
|----------|------|
| **血液検査ツール** | 純粋なフロントエンドJavaScript計算 — 入力データはブラウザの外に出ず、サーバーに送信されることもありません |
| **AIチャット** | 会話はEdge Functionでリアルタイム処理 — 保存、記録、トレーニングへの使用は一切なし |
| **アナリティクス** | Umamiセルフホスト分析 — ページビューのみ集計、IPアドレスや個人識別情報は収集しません |
| **トラッキング** | サードパーティのトラッキングスクリプトはゼロ — Google Analyticsなし、SNSトラッカーの埋め込みなし |
| **Cookie** | テーマ設定（ダーク/ライトモード）のみに使用、トラッキングには使用しません |
| **広告** | 広告なし、今後も掲載しません |
