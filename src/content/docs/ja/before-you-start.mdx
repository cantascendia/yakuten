## <span style="font-family: var(--font-mono); color: var(--color-accent);">§ 1.1</span> ベースライン血液検査

以下の検査はHRT開始前に実施する必要があります。内分泌科、ジェンダークリニック、または一般的なクリニックで検査を受けることができます。日本では、GID（性同一性障害）学会認定医療機関やジェンダー外来を設置する大学病院で対応可能です。

## <span style="font-family: var(--font-mono); color: var(--color-danger);">§ 1.2</span> 絶対的禁忌

<ContraindicationBox type="absolute" title="絶対的禁忌" sectionNum="">

- 活動性の血栓塞栓症（例：深部静脈血栓症、肺塞栓症）
- ホルモン感受性悪性腫瘍の確定診断または疑い（例：乳がん、前立腺がん）
- 急性期の重度肝障害または腎障害
- 原因不明の性器出血（トランスマスキュリンの方の場合）
- 薬剤成分に対する重篤なアレルギーの既往

</ContraindicationBox>

## <span style="font-family: var(--font-mono); color: var(--color-caution);">§ 1.3</span> 相対的禁忌

<ContraindicationBox type="relative" title="相対的禁忌" sectionNum="">

- コントロール不良の高血圧または心血管疾患
- コントロール不良の糖尿病
- 喫煙（特に35歳以上。VTEリスクが著しく増加します）
- 胆嚢疾患の既往
- 前兆を伴う片頭痛（慎重なリスク評価が必要）
- 重度肥満（BMI &gt; 30）

</ContraindicationBox>
