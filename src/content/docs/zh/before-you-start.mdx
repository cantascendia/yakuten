## <span style="font-family: var(--font-mono); color: var(--color-danger);">§ 1.2</span> 绝对禁忌症 (Absolute)

<ContraindicationBox type="absolute" title="绝对禁忌症" sectionNum="">

- 现患血栓栓塞性疾病（如深静脉血栓、肺栓塞）
- 确诊或疑似激素依赖性肿瘤（如乳腺癌、前列腺癌）
- 严重肝肾功能不全或处于急性期
- 不明原因的阴道出血（针对跨性别男性）
- 已知对药物成分严重过敏

</ContraindicationBox>

## <span style="font-family: var(--font-mono); color: var(--color-caution);">§ 1.3</span> 相对禁忌症 (Relative)

<ContraindicationBox type="relative" title="相对禁忌症" sectionNum="">

- 未控制的高血压或心血管疾病
- 控制不佳的糖尿病
- 吸烟（尤其是35岁以上，显著增加血栓风险）
- 胆囊疾病史
- 偏头痛（先兆型偏头痛需谨慎评估）
- 严重肥胖 (BMI &gt; 30)

</ContraindicationBox>
