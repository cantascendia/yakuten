## 复审团队结构

HRT药典的医学内容由跨专业志愿团队复审,按主题分派:

| 领域 | 复审职责 |
|---|---|
| 内分泌科 | 雌激素/抗雄/孕激素剂量、血检解读、代谢与骨密度 |
| 精神科 | 性别肯定评估、情绪波动、合并用药相互作用 |
| 肝病 / 血液科 | CPA 肝毒性、VTE 风险、凝血监测 |
| 法律顾问 | 中国大陆处方合规、隐私法规、国际旅行携药 |

## 复审原则

- **无处方关系**:复审人与本站推荐的药物/品牌无直接处方开具关系,避免利益冲突
- **异议归档**:复审意见与编辑决定全程记录,重大分歧公开披露
- **身份保护**:顾问可选择匿名,但必须向编辑部出示执业资质复印件与所在机构证明
- **更新义务**:指南重大更新后 48 小时内,复审人确认是否影响现有页面
