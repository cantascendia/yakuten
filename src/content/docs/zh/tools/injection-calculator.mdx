选择你使用的注射用雌二醇规格，输入目标周剂量，即可获取每次注射的抽取体积、推荐注射器型号和预期雌二醇（E2）血药浓度范围。

## 使用说明

1. **选择药物规格** — 从下拉菜单选择你所使用的注射剂，默认为 Progynon Depot 10mg/mL
2. **输入目标剂量** — 通过数字框或滑块设置每周注射剂量（0.5\u201310 mg）
3. **查看结果** — 工具会实时显示抽取体积、注射器型号建议、预期 E2 范围及适用人群

:::caution[免责声明]
本工具仅供参考，不构成医疗建议。预期 E2 范围为近似谷值估算，实际血药浓度因个体差异显著不同。请务必在医疗专业人员指导下确定和调整剂量，并定期进行血检监测。
:::
