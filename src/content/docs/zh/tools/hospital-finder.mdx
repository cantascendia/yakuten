## 找不到本地资源？

<GlassCard>

**三个替代方案：**

1. **任何三甲医院内分泌科**
   你不必挂专门的跨性别门诊。内分泌科医生可以评估激素水平、排查基础疾病。就诊时说"内分泌有点不稳定，想检查激素水平"即可，无需透露跨性别身份。→ [详细说明](/zh/china-reality/#-84-不用告诉医生你是跨性别也能做检查)

2. **跨性别社区本地群组**
   各地社群通常维护有更新的本地医疗资源列表。可以通过 TRANS.CN、跨儿心理、彩虹暴风雪等组织的社交媒体寻找本地群组。

3. **互联网医院 + 血检结合**
   部分互联网医院平台（如平安健康、微医）有接受跨性别患者的医生，结合本站[血检工具](/zh/tools/blood-checker/)进行自我监测。

</GlassCard>

## 关于数据更新

医疗资源信息会随时间变化。如果你有新信息、发现数据已过时，或希望推荐其他友好医生/科室，欢迎通过以下方式反馈：

- GitHub Issues（如果你熟悉 GitHub）
- 本站联系邮件（见页脚）

我们计划每季度审核并更新一次数据。
