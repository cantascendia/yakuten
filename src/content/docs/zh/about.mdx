## 使命

**循证 · 减害 · 引导就医**

HRT药典是面向中文圈跨性别女性的 HRT 安全底线信息站。我们的目标不是替代医生，而是帮助已在用药或即将用药者：

- **看懂自己的血检报告**，知道哪些指标异常需要就医
- **了解药物的真实风险**，而非社区传言和未经验证的经验
- **识别紧急危险信号**（DVT、肝损伤、脑膜瘤），在关键时刻做出正确决策
- **找到友好的医疗资源**，从 DIY 走向规范化医疗

### 本站不是什么

- ❌ 不是百科全书或医学教科书
- ❌ 不是论坛、社群或交流平台
- ❌ 不是购药渠道或药品推荐站
- ❌ 不提供个人化处方建议（"你应该吃 X mg"）

## 内容来源

本站所有医疗内容基于以下国际临床指南和同行评审文献：

| 来源 | 说明 |
|------|------|
| **Endocrine Society 2017** | 《Endocrine Treatment of Gender-Dysphoric/Gender-Incongruent Persons》临床实践指南 |
| **WPATH SOC 8 (2022)** | 《Standards of Care for the Health of Transgender and Gender Diverse People》第 8 版 |
| **UCSF Transgender Care** | 加州大学旧金山分校跨性别护理指南 |
| **EMA (2020)** | 欧洲药品管理局关于 CPA 的限制令和安全评估 |
| **PubMed / PMC** | 同行评审的原始研究论文（每条引用附 DOI） |

每条医疗建议旁的上标数字（如 <sup>[1]</sup>）均链接至具体文献。**没有引用 = 没有内容** — 这是本站的铁律。

## 内容审核流程

| 环节 | 标准 |
|------|------|
| **引用要求** | 每条临床建议至少追溯至 1 篇同行评审文献或国际指南 |
| **交叉验证** | 药物剂量范围必须与 ≥2 个独立来源交叉验证（如 WPATH + Endocrine Society + UCSF） |
| **证据分级** | 每页标注证据等级：A（多项 RCT/Meta）、B（单项 RCT/队列）、C（病例/专家意见）、X（无证据） |
| **审核日期** | 每页底部标注最后审核日期 |
| **更新周期** | 每季度审查一次内容时效性，根据新发表文献更新 |

<WarningBox title="内容局限性">
本站内容团队目前为个人维护，尚未经过专业医学编辑的正式同行评审。我们尽最大努力确保准确性，但错误和遗漏是可能的。如果你发现任何不准确之处，欢迎通过 GitHub 提交 Issue。
</WarningBox>

## 技术实现

| 技术 | 用途 |
|------|------|
| **Astro + Starlight** | 静态站点生成，默认零 JS，加载极快 |
| **React Islands** | 仅交互工具（血检自查、剂量模拟器等）使用客户端 JS |

| **Vercel** | 静态托管 + Edge Functions（AI 问答） |
| **开源** | 完整源代码公开于 [GitHub](https://github.com/cantascendia/yakuten) |

## 隐私保障

**本站不收集、不存储任何个人数据。**

| 类别 | 措施 |
|------|------|
| **血检工具** | 纯前端 JavaScript 计算，输入的数据不离开你的浏览器，不发送到任何服务器 |
| **AI 问答** | 对话内容通过 Edge Function 实时处理，不存储、不记录、不用于训练 |
| **分析** | 使用 Umami 自托管分析，仅统计页面访问量，不采集 IP 地址或个人标识 |
| **追踪** | 零第三方追踪脚本，不使用 Google Analytics、不嵌入社交媒体追踪器 |
| **Cookies** | 仅用于主题偏好（深色/浅色模式），不用于追踪 |
| **广告** | 无广告，永远不会有 |

## 免责声明

本站内容**仅供教育和参考用途**，不构成医疗建议。

- 本站不能替代合格医生的面对面诊断和治疗
- 药物的使用、调整和停用应在有资质的医疗专业人员指导下进行
- 如出现任何紧急症状（单侧腿肿痛、突发胸痛、黄疸等），请立即停药并前往最近的急诊
- 本站不对因参考本站内容而产生的任何医疗后果承担责任

**自我用药（DIY）存在真实风险。** 本站的目标是帮助你更安全地用药，但最安全的方式始终是在医生指导下进行 HRT。
