# 发布就绪检查

## 功能层面
- [ ] 所有计划功能已实现并通过用户验收（非占位/mock）
- [ ] 核心用户流程端到端测试全部通过
- [ ] 已知 bug 列表中无 Critical 和 Major 级别未修复项
- [ ] 空状态、加载态、错误态、边界情况已全部处理

## 技术层面
- [ ] CI 流水线全绿（lint + test + build）
- [ ] 代码中无 TODO / FIXME / HACK 残留（`grep -rn "TODO\|FIXME\|HACK" lib/`）
- [ ] 环境配置已切换到生产（API 地址、密钥、功能开关）
- [ ] 代码混淆 / ProGuard 已配置（如适用）
- [ ] App 签名证书正确
- [ ] 最低系统版本要求已确认（`minSdkVersion` / `MinimumOSVersion`）
- [ ] 依赖版本锁定（`pubspec.lock` / `package-lock.json` 已提交）

## 应用商店合规
- [ ] App 图标和启动画面为正式版（非占位图）
- [ ] 各尺寸商店截图已准备
- [ ] 应用描述、关键词、分类已填写
- [ ] 隐私政策 URL 已上线且可访问
- [ ] 权限使用说明已填写（摄像头、定位、通知、麦克风等的用途）
- [ ] 年龄分级已填写
- [ ] 内购/订阅（如有）在商店后台已配置
- [ ] Apple 审核指南自查：无私有 API、无隐藏功能、无误导性描述
- [ ] Google Play 政策自查：数据安全声明已填写、目标 API 级别达标

## 可观测性
- [ ] 崩溃监控 SDK 已集成且验证（手动触发一次 crash 确认上报）
- [ ] 关键埋点已部署且验证（检查事件出现在分析后台）
- [ ] 性能基线已记录（冷启动时间、核心页面加载时间）
